# Release v0.4.1

## 新功能

### Homebrew 安装支持

现在可以通过 Homebrew 安装 gcop-rs：

```bash
brew tap AptS-1547/gcop-rs
brew install gcop-rs
```

支持 macOS (Intel/Apple Silicon) 和 Linux (x86_64/ARM64)。

### cargo-binstall 支持

支持通过 cargo-binstall 快速安装预编译二进制：

```bash
cargo binstall gcop-rs
```

无需编译，直接下载对应平台的二进制文件。

## 安装方式汇总

```bash
# Homebrew (macOS/Linux)
brew tap AptS-1547/gcop-rs
brew install gcop-rs

# cargo-binstall (推荐，无需编译)
cargo binstall gcop-rs

# cargo install (从源码编译)
cargo install gcop-rs

# 预编译二进制
# 从 Releases 页面下载对应平台的二进制文件
```

## 升级说明

从 v0.4.0 升级**无需任何操作**，完全向后兼容。

## 安装

```bash
cargo install gcop-rs
```

或从 [Releases](https://github.com/AptS-1547/gcop-rs/releases) 下载预编译二进制。
