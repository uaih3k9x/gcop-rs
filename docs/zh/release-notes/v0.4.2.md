# Release v0.4.2

## 新功能

### PyPI 安装支持

现在可以通过 pipx 或 pip 安装 gcop-rs：

```bash
# 使用 pipx（推荐）
pipx install gcop-rs

# 使用 pip
pip install gcop-rs
```

这是一个 Python 包装器，首次运行时会自动从 GitHub Releases 下载对应平台的预编译 Rust 二进制文件。

### CLI 帮助彩色输出

`--help` 输出现在带有颜色高亮，提升可读性：
- 标题（Usage、Commands 等）显示为绿色加粗
- 命令和选项名显示为青色加粗

## 安装方式汇总

```bash
# Homebrew (macOS/Linux)
brew tap AptS-1547/gcop-rs
brew install gcop-rs

# pipx (Python 用户，推荐)
pipx install gcop-rs

# cargo-binstall (无需编译)
cargo binstall gcop-rs

# cargo install (从源码编译)
cargo install gcop-rs
```

## 升级说明

从 v0.4.1 升级**无需任何操作**，完全向后兼容。

## 安装

```bash
cargo install gcop-rs
```

或从 [Releases](https://github.com/AptS-1547/gcop-rs/releases) 下载预编译二进制。
