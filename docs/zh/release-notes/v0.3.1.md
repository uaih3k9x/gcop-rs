# Release v0.3.1

## 平台支持扩展

本次更新主要扩展了 CI 构建平台支持，现在提供更多预编译二进制：

| 平台 | 架构 | 状态 |
|------|------|------|
| Linux | x86_64 | ✅ |
| Linux | aarch64 (ARM64) | ✅ **新增** |
| macOS | x86_64 (Intel) | ✅ 恢复 |
| macOS | aarch64 (Apple Silicon) | ✅ |
| Windows | x86_64 | ✅ |
| Windows | aarch64 (ARM64) | ✅ **新增** |

> 现在 Linux ARM64 用户（如树莓派 64 位、AWS Graviton）和 Windows ARM64 用户可以直接下载预编译版本了。

## 依赖优化

- **git2 依赖瘦身**：禁用 git2 默认特性，移除 openssl 相关依赖
  - 简化依赖树，减少编译时间
  - 提升跨平台构建兼容性
  - 解决部分平台上 openssl 链接问题

## 文档更新

- README 新增 `gcop config edit` 命令使用说明

## 升级说明

从 v0.3.0 升级**无需任何操作**，完全向后兼容。

## 安装

```bash
cargo install gcop-rs
```

或从 [Releases](https://github.com/AptS-1547/gcop-rs/releases) 下载预编译二进制。
