# Release v0.1.0

gcop-rs 首次发布 - AI 驱动的 Git 提交信息生成器和代码审查工具。

## 🎯 核心功能

- **AI 提交信息** - 使用 Claude、OpenAI 或 Ollama 生成符合规范的提交信息
- **AI 代码审查** - 获取变更的安全性和性能建议
- **交互式工作流** - 接受、编辑、带反馈重试或退出
- **Git 别名** - 11 个便捷快捷方式（`git c`、`git r`、`git acp` 等）
- **美观 CLI** - 彩色输出、加载动画和交互式菜单

## 📦 安装

```bash
cargo install gcop-rs
```

或下载适用于您平台的预编译二进制文件。

## 🚀 快速开始

```bash
# 初始化配置
gcop-rs init

# 生成提交信息
git add .
git c

# 审查变更
git r
```

## 🔧 包含内容

### 命令
- `init` - 交互式配置向导
- `commit` - AI 提交信息生成
- `review` - AI 代码审查
- `config` - 配置管理
- `alias` - Git 别名管理

### Git 别名
- `git c` - AI 提交
- `git r` - AI 审查
- `git ac` - 添加全部并提交
- `git acp` - 添加、提交并推送
- `git undo` - 撤销上次提交（保留暂存）
- `git p` / `git pf` - 推送快捷方式
- 等等...

### 支持的 LLM 提供商
- Claude (Anthropic)
- OpenAI
- Ollama (本地)
- 任何 OpenAI/Claude 兼容 API
